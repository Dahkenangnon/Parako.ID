# Security Policy

## 🚨 Important Notice

_Parako.ID is currently in active development and is NOT recommended for production by now_

This project is under active development and may contain security vulnerabilities or incomplete features. We strongly recommend against using this software in production environments until the first stable release.

## 🛡️ Security Philosophy

**Parako.ID follows a security-first approach by minimizing external dependencies and prioritizing Node.js built-in APIs wherever possible to reduce attack surface and maintain better control over security-critical code.**

## 🔒 Reporting Security Issues

If you discover a security vulnerability in Parako.ID, please report it immediately:

**📧 Email**: [dah.kenangnon@gmail.com](mailto:dah.kenangnon@gmail.com)

### What to Include

- Clear description of the vulnerability
- Steps to reproduce the issue
- Potential impact assessment
- Environment details (version, OS, configuration)

### Response Timeline

- **Initial Response**: Within 48 hours
- **Resolution**: As quickly as possible based on severity
