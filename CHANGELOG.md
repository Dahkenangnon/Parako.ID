# Changelog

All notable changes to this project are documented in this file. The format is based on Conventional Commits, with sections derived from commit types (`feat` → Features, `fix` → Fixes, `refactor` → Refactor, `docs` → Documentation, etc.). Maintenance-only commits (`chore`, `ci`, `style`, `test`) are intentionally hidden from public release notes.

Releases are produced automatically by [semantic-release](https://semantic-release.gitbook.io/) on every push to `main`. See the [GitHub Releases page](https://github.com/Dahkenangnon/Parako.ID/releases) for the same content plus signed artifacts.
